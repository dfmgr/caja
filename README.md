## caja  
  
File Manager  
  
Automatic install/update:

```shell
bash -c "$(curl -LSs https://github.com/dfmgr/caja/raw/main/install.sh)"
```

Manual install:
  
requires:

Debian based:

```shell
apt install caja
```  

Fedora Based:

```shell
yum install caja
```  

Arch Based:

```shell
pacman -S caja
```  

MacOS:  

```shell
brew install caja
```
  
```shell
mv -fv "$HOME/.config/caja" "$HOME/.config/caja.bak"
git clone https://github.com/dfmgr/caja "$HOME/.config/caja"
```
